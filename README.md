# fourvoice-captioner-team_enclaive
Four-style AI video captioning (formal, sarcastic, humorous-tech, humorous-non-tech) for Track 2 of the AMD Act 2 x lablab.ai hackathon. Grounds captions in real on-screen visuals and audio via Fireworks AI, with a shared fact-checked source across all styles and a confidence score based on verified evidence.
